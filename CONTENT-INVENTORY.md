## 🔄 How to Update Content

### Through Decap CMS (Recommended)
1. Visit `https://stmarkchristian.school/admin`
2. Log in with your credentials
3. Navigate to the collection you want to edit
4. Click "Edit" or "New [Content Type]"
5. Make your changes
6. Click "Save" and then "Publish"

### Direct File Editing
1. Navigate to `src/content/[collection]/`
2. Open the `.md` file you want to edit
3. Update the frontmatter (between `---`) or content
4. Save the file
5. Commit and push to GitHub
6. Changes will deploy automatically

## 📝 Content Guidelines

### News & Stories
- **Recommended length:** 400-800 words
- **Images:** 1200x800px recommended
- **Categories:** Community, Admissions, Academics, Faith & Formation, Programs
- **Best practices:** Include Scripture, real stories, call-to-action

### Testimonials
- **Recommended length:** 150-300 words
- **Format:** First-person, authentic voice
- **Include:** Name, role/grade of children, specific benefits
- **Images:** Family photos or professional headshots

### Events
- **Required fields:** Title, date, time, location, category
- **Include:** Clear schedule, RSVP instructions, what to bring
- **Registration links:** Use actual URLs or `/admissions/inquire`

## 🎨 Image Placeholders

All content currently uses placeholder image paths like `/uploads/[name].jpg`. To add real images:

1. **Upload through CMS:**
   - Edit a post in `/admin`
   - Click "Choose an image"
   - Upload your file
   - It will automatically save to `/public/uploads/`

2. **Manual upload:**
   - Place images in `/public/uploads/`
   - Reference as `/uploads/filename.jpg` in content
