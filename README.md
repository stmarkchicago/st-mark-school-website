# St. Mark Christian School Website

A modern, content-managed website built with Astro and deployed to Cloudflare Workers.

## 🎉 New Feature: Content Management System

This site now includes **Decap CMS** - a user-friendly admin panel for managing all your content!

### Quick Start

```bash
# Terminal 1: Start CMS backend
npm run cms

# Terminal 2: Start dev server
npm run dev
```

Then open: **http://localhost:3000/admin/**
