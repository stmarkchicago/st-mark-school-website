## 🚀 Getting Started (Local)

**Terminal 1:**
```bash
npm run cms
```
(Keep running!)

**Terminal 2:**
```bash
npm run dev
```

**Then open:** `http://localhost:3000/admin/`

---

## ✏️ How to Add Content

### News Post
1. Click **"News & Announcements"**
2. Click **"New"**
3. Fill in title, date, summary, body
4. Upload image (optional)
5. Click **"Publish"**

### Event
1. Click **"Events"**
2. Click **"New"**
3. Enter event name, dates, location
4. Add description
5. Click **"Publish"**

### Testimonial
1. Click **"Testimonials"**
2. Click **"New"**
3. Enter name, role, quote
4. Upload photo (optional)
5. Check "Display on Homepage" if featured
6. Click **"Publish"**

---

## 🎨 Edit Homepage

1. Click **"Site Settings"**
2. Click **"Homepage"**
3. Edit hero heading/subheading
4. Edit button text/links
5. Edit identity bar taglines
6. Click **"Publish"**

---

## 📞 Edit Contact Info

1. Click **"Site Settings"**
2. Click **"Contact Information"**
3. Update phone, email, address
4. Update office hours
5. Click **"Publish"**

---

## 📸 Upload Images

### In Any Image Field:
1. Click **"Choose an image"**
2. **Drag & drop** photo OR click to browse
3. Image uploads automatically
4. Click **"Insert"**

### Tips:
- ✅ Use 1200x800px size
- ✅ Keep under 5MB
- ✅ Use .jpg or .png
- ✅ Use descriptive filenames

---

## 💾 Save vs Publish

| Button | What It Does |
|--------|--------------|
| **Save Draft** | Saves without publishing (not live) |
| **Publish** | Makes content live on website |
| **Delete** | Removes content permanently |

| Button | What It Does |
|--------|--------------|
| **B** | Bold text |
| **I** | Italic text |
| **H1/H2/H3** | Headings (large, medium, small) |

| **1.** | Numbered list |

## ✅ Best Practices

### Do's ✅
- ✅ Write clear, descriptive titles
- ✅ Use high-quality images
- ✅ Save drafts frequently
- ✅ Preview before publishing
- ✅ Fill in all required fields (*)
- ✅ Use proper grammar/spelling

### Don'ts ❌
- ❌ Don't upload huge images (>5MB)
- ❌ Don't forget to click "Publish"
- ❌ Don't leave required fields empty
- ❌ Don't use copyrighted images

---

## 🆘 Troubleshooting

| Problem | Solution |
|---------|----------|
| Can't access /admin/ | Check both terminals are running |
| Changes not showing | Did you click "Publish"? |
| Image won't upload | Check file size (<5MB) and format |
| Forgot to save | Changes are auto-saved as you type |
| Made a mistake | Edit the item again and republish |

## 🎯 Quick Tips

💡 **Tip #1:** Write your content in Word first, then copy/paste into CMS

💡 **Tip #2:** Take screenshots of your published content for reference

💡 **Tip #3:** Create a "Content Calendar" to plan posts in advance

💡 **Tip #4:** Keep a folder of pre-approved images ready to upload

💡 **Tip #5:** Review and remove old events/news regularly
