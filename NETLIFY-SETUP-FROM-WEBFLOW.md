# Netlify Setup Guide - From Webflow Cloud

This guide will help you set up Netlify CMS so you can edit content directly from your website's `/admin` panel.

## What We're Doing

1. Getting your code from Webflow Cloud to GitHub
2. Connecting GitHub to Netlify
3. Deploying to Netlify with CMS enabled

---

## Step 1: Contact Webflow Support

Since you're working in Webflow Cloud, you need to get your project files. Here's what to do:

### Email Webflow Support:

**To:** support@webflow.com

**Subject:** Need to export App project code to GitHub

**Message:**
```
Hi Webflow Support,

I'm working on an App project in Webflow Cloud (Project: St. Mark School Site, ID: 93539ce4-06ad-45d8-859b-ba49ccae586d).

I need to deploy this app to Netlify to enable the CMS admin panel. Can you help me:

1. Export the complete project code, OR
2. Set up GitHub integration for this app project, OR
3. Provide access to download the project files

The app is currently deployed at: new.stmarkchristian.school/app

Thank you!
```

---

## Step 2: Once You Have the Files

When Webflow provides you with the files or GitHub integration, we'll:

1. **Push code to GitHub** (if not already there)
2. **Connect to Netlify**
3. **Configure the CMS**
4. **Deploy**

---

## Alternative: Manual File Transfer

If Webflow support can't help quickly, let me know and I'll provide you with another method to get the files.

---

## Questions?

Let me know what Webflow support says, and we'll proceed from there!
